# 18PlusList.txt 文件说明

## 文件作用
这是一个 18+ 网站域名黑名单文件，用于在导入书源时过滤和阻止导入成人内容网站的书源。

## 文件格式
- 原始文件：每行是一个 Base64 编码的域名
- 解码后：每行是一个域名或网站标识符

## 统计信息
- 原始条目数：284 条
- 去重后条目数：207 条
- 解码文件：`18PlusList_decoded.txt`
- 去重文件：`18PlusList_decoded_unique.txt`

## 使用方法
在导入书源时，需要：
1. 读取 `assets/18PlusList.txt` 文件
2. 解码所有 Base64 编码的域名
3. 检查待导入书源的 URL 是否在黑名单中
4. 如果在黑名单中，拒绝导入并提示用户："xxx是18+网址,禁止导入."
