# 跨平台图标使用说明

## 概述

从参考项目复制的 Android 图标资源（PNG 格式）**可以用于其他平台**，但需要根据各平台的要求进行转换和放置。

## 平台图标要求对比

### Android
- ✅ **已完成** - 已从参考项目复制所有尺寸
- 格式：PNG
- 位置：`android/app/src/main/res/mipmap-*/ic_launcher.png`
- 尺寸：48x48, 72x72, 96x96, 144x144, 192x192

### Web
- ⚠️ **需要更新** - 当前有默认图标，可以使用 Android 图标替换
- 格式：PNG
- 位置：`web/icons/`
- 需要尺寸：192x192, 512x512
- 建议使用：`mipmap-xxhdpi/ic_launcher.png` (144x144) 和 `mipmap-xxxhdpi/ic_launcher.png` (192x192)

### macOS
- ⚠️ **需要更新** - 当前有默认图标，可以使用 Android 图标替换
- 格式：PNG
- 位置：`macos/Runner/Assets.xcassets/AppIcon.appiconset/`
- 需要尺寸：16x16, 32x32, 64x64, 128x128, 256x256, 512x512, 1024x1024
- 建议使用：`mipmap-xxxhdpi/ic_launcher.png` (192x192) 作为基础，需要放大到 1024x1024

### iOS（如果支持）
- ⚠️ **需要创建** - 如果项目支持 iOS
- 格式：PNG
- 位置：`ios/Runner/Assets.xcassets/AppIcon.appiconset/`
- 需要尺寸：多种尺寸（20x20 到 1024x1024）
- 建议使用：`mipmap-xxxhdpi/ic_launcher.png` (192x192) 作为基础

## 图标转换建议

### 方法 1：直接使用（适合 Web）

Web 平台可以直接使用 Android 的 PNG 图标：

```bash
# 使用 192x192 的图标作为 Web 图标
cp android/app/src/main/res/mipmap-xxxhdpi/ic_launcher.png web/icons/Icon-192.png
cp android/app/src/main/res/mipmap-xxxhdpi/ic_launcher.png web/icons/Icon-512.png
```

### 方法 2：使用工具转换（适合 macOS/iOS）

可以使用以下工具将 Android 图标转换为其他平台格式：

1. **ImageMagick**（命令行）
   ```bash
   # 调整大小
   convert android/app/src/main/res/mipmap-xxxhdpi/ic_launcher.png -resize 1024x1024 macos/Runner/Assets.xcassets/AppIcon.appiconset/app_icon_1024.png
   ```

2. **在线工具**
   - [AppIcon.co](https://www.appicon.co/) - 自动生成所有平台图标
   - [IconKitchen](https://icon.kitchen/) - Google 的图标生成工具

3. **Flutter 插件**
   - `flutter_launcher_icons` - 自动生成所有平台图标

## 推荐方案：使用 flutter_launcher_icons

最简单的方式是使用 `flutter_launcher_icons` 插件，它会自动从单个源图标生成所有平台的图标。

### 安装步骤

1. 添加依赖到 `pubspec.yaml`：
```yaml
dev_dependencies:
  flutter_launcher_icons: ^0.13.1
```

2. 配置图标源：
```yaml
flutter_launcher_icons:
  android: true
  ios: true
  web:
    generate: true
    image_path: "assets/images/app_icon.png"
  macos:
    generate: true
    image_path: "assets/images/app_icon.png"
  image_path: "android/app/src/main/res/mipmap-xxxhdpi/ic_launcher.png"
```

3. 运行生成命令：
```bash
flutter pub get
flutter pub run flutter_launcher_icons
```

## 当前状态

- ✅ **Android** - 已从参考项目复制，完整配置
- ⚠️ **Web** - 有默认图标，可以使用 Android 图标替换
- ⚠️ **macOS** - 有默认图标，可以使用 Android 图标替换
- ❓ **iOS** - 如果项目支持，需要创建图标资源

## 建议操作

1. **立即可以做的**：将 Android 的 192x192 图标复制到 Web 平台
2. **推荐做的**：使用 `flutter_launcher_icons` 插件统一管理所有平台图标
3. **可选做的**：手动调整图标尺寸以适配 macOS/iOS 的要求

## 注意事项

- Android 图标是 PNG 格式，可以直接用于 Web
- macOS/iOS 需要更多尺寸，建议使用工具自动生成
- 保持所有平台图标设计一致，提升用户体验
- 如果使用自适应图标（Android 8.0+），需要确保图标在圆形和方形遮罩下都显示良好
